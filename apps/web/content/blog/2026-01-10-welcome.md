# Welcome to Infinity Globalindo Blog

This is a sample blog post created to demonstrate the Decap CMS integration. You can edit this post or create new ones using the admin interface at `/admin/`.

## Features

- **Markdown Support**: Write your content in markdown format
- **Git-Based**: All content is stored in your repository
- **Easy Editing**: User-friendly interface for content management
- **Version Control**: Every change is tracked in git

## Getting Started

1. Navigate to `/admin/` to access the CMS
2. Create, edit, or delete blog posts
3. Your changes will be saved to the repository

Happy blogging!
